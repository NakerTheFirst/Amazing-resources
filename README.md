# Amazing-resources
A personalised(!) amazing list of resources: Data Science, Computer Science and technology

- [Michael Sambol's YT channel](https://www.youtube.com/c/MichaelSambol): Algorithms and Data Structures visually explained in 3* minutes (varies) 
- [Big O Cheat Sheet](https://www.bigocheatsheet.com/): The go-to time/space complexity cheat sheet of common Data Structures and Algorithms
- [Data Lumina DS Project Template](https://github.com/datalumina/datalumina-project-template): A robust Python template for Data Science projects based on Cookiecutter Data Science Project Structure
- [Minimal DS Project Template](https://github.com/NakerTheFirst/DS-project-template): A minimal Python template for Data Science projects based on Data Lumina's template
- [Thomas H. Cormen et al.'s Introduction to Algorithms book](https://www.wikiwand.com/en/articles/Introduction_to_Algorithms): The holy bible of Algorithms and Data Structures
- [FreeCodeCamp's 1h Git Course](https://www.youtube.com/watch?v=RGOj5yH7evk): A beginner friendly
